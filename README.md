A simple Python tool that automatically organizes files into folders based on file type.

## Features

- Automatically sorts files
- Creates folders if they don't exist
- Supports images, videos, documents, and music

## How to Run

1. Clone this repository
2. Run:

python main.py

## Example

Before:
test/
photo.jpg
song.mp3
file.pdf
video.mp4

After:
test/
Images/photo.jpg
Music/song.mp3
Documents/file.pdf
Videos/video.mp4
